# 早盘资金雷达

## 目标

早盘资金雷达用于短线研究：快速观察今天大资金更偏向哪些板块、哪些板块流出最多、哪些个股出现急拉。

它回答：

- 主力净流入靠前的板块是谁？
- 主力净流出靠前的板块是谁？
- 哪些个股涨速靠前，并伴随资金流入？
- 当前更像短线进攻、分歧确认，还是应继续观察？

它不回答：

- 立刻买入哪只股票。
- 明天一定上涨。
- 某个板块一定持续。

## 数据边界

默认实现为 `MorningMoneyRadarClient`：

- 优先请求公开行情接口；
- 解析板块主力净流入、主力净流出、个股涨速和资金字段；
- 如果接口失败或字段变化，返回 `offline_sample` 并标注 `data_status: sample`；
- 所有输出都带 `as_of`、`source` 和 `data_status`。

## 输出结构

```json
{
  "data_status": "real_time",
  "market_phase": "早盘主升观察",
  "top_inflow_sectors": [],
  "top_outflow_sectors": [],
  "fast_movers": [],
  "shortline_read": "早盘资金最强方向暂看...",
  "risks": []
}
```

## 短线使用规则

仅当以下条件同时改善时，短线战法才可以从“观察”转入“候选”：

- 板块主力净流入持续靠前；
- 龙头股急拉后不快速回落；
- 板块内多只股票同步放量；
- 市场情绪周期不在退潮；
- 个股没有公告、ST、流动性等硬风险；
- 符合用户自己的止损和持仓周期。

早盘急拉最容易出现冲高回落，因此雷达输出默认只作为观察清单。
